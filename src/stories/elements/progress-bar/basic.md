# Progress Bar

### Usage

```js
<dp-progress-bar
  :percent="percent"
  :height="height"
  :indicatorOnMobile="indicatorOnMobile"
  :indicatorOnDesktop="indicatorOnDesktop">
</dp-progress-bar>
```
